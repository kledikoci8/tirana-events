🚀 DEPLOYMENT INSTRUCTIONS
═══════════════════════════════════════════════════════════════

BACKEND DEPLOYMENT:

1. Set Environment Variables:
   export JWT_SECRET="your-super-secure-256-bit-secret-here"
   export DB_USERNAME="tirana_events_user"
   export DB_PASSWORD="your-secure-db-password"
   export CORS_ALLOWED_ORIGINS="https://tiranaevents.com,https://app.tiranaevents.com"

2. Build:
   cd backend
   mvn clean package -DskipTests

3. Run:
   java -jar target/events-backend-1.0.0.jar

FRONTEND DEPLOYMENT:

1. Update API URL:
   Edit: mobile/src/config/api.js
   Set prod.apiUrl to your production backend URL

2. Build:
   cd mobile
   eas build --platform ios
   eas build --platform android

3. Deploy:
   eas submit --platform ios
   eas submit --platform android
