🚀 SETUP INSTRUCTIONS
═══════════════════════════════════════════════════════════════

BACKEND SETUP:
────────────────────────────────────────────────────────────────
1. Run database migrations:
   mysql -u root -p tirana-events < backend/database-migrations.sql

2. Add Firebase service account:
   - Download firebase-service-account.json from Firebase Console
   - Place in: backend/src/main/resources/

3. Update application.properties:
   - Set your database credentials
   - Add Firebase configuration (optional)

4. Build and run:
   cd backend
   ./mvnw clean install
   ./mvnw spring-boot:run

FRONTEND SETUP:
────────────────────────────────────────────────────────────────
1. Install dependencies:
   cd mobile
   npm install

2. Update API URL in services:
   - Edit: mobile/src/services/personalizationService.js
   - Change: const API_URL = 'http://YOUR_IP:8080/api';

3. Run the app:
   npm start
   # Then press 'i' for iOS or 'a' for Android
